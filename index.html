AI

I recently created an AI sports betting bot using Python and the OpenAI API. The bot makes api calls to get the latest odds of NFL games, then uses the OpenAI API to analyze the data and make bets based on predicted outcomes. The API call and portions of the analysis were done by providing tools to the LLM and having it decide when to use them.
I have used AI LLMs like ChatGPT and Copilot for use in programming. The best use has been using it to help understand and resolve errors. I have spent time learning how to effectively use AI to improve my skills and understanding
I do feel it is important to never rely on the technology. I review each line of code written by AI and do not use it to do things I do not understand. I make sure that AI is only used to improve the speed of my work rather than replacing learning or practice with it.
This semester, I am taking a course on AI, LLMs, and Machine Learning.
